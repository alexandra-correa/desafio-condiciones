# Desafío Condiciones

Proyecto desarrollado con HTML, CSS y JavaScript, enfocado en la aplicación de estructuras condicionales, manejo de eventos y manipulación del DOM mediante ejercicios interactivos.

---

## Enlaces del proyecto

- Repositorio: https://github.com/alexandra-correa/desafio-condiciones
- Deploy: https://alexandra-correa.github.io/desafio-condiciones/

---

## Descripción

Este proyecto fue desarrollado con el objetivo de poner en práctica conceptos fundamentales de JavaScript, tales como:

- Estructuras condicionales (`if`, `else`, `switch`).
- Manejo de eventos con `addEventListener`.
- Manipulación del DOM.
- Captura y validación de datos ingresados por el usuario.
- Actualización dinámica de elementos HTML.

El desafío se encuentra dividido en tres ejercicios independientes.

---

## Ejercicio 1: Cambio de borde en imagen

Se desarrolló una funcionalidad que permite agregar y quitar un borde a una imagen mediante un evento de clic.

### Funcionalidades implementadas

- Detección de eventos utilizando `addEventListener`.
- Manipulación dinámica de clases CSS.
- Uso de métodos `classList.add()`, `classList.remove()` y `classList.contains()`.
- Actualización visual de la interfaz de usuario.

### Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## Ejercicio 2: Sistema de selección de stickers

Se implementó un sistema de selección de stickers que permite al usuario elegir cantidades y validar que el total no exceda el máximo permitido.

### Funcionalidades implementadas

- Captura de valores desde inputs numéricos.
- Conversión de datos utilizando `Number()`.
- Validación mediante estructuras condicionales.
- Manipulación dinámica del contenido del DOM.

### Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

---

## Ejercicio 3: Validación de password

Se desarrolló un sistema de validación de contraseñas utilizando elementos `select` y lógica condicional.

### Funcionalidades implementadas

- Obtención de valores seleccionados por el usuario.
- Construcción dinámica de la contraseña.
- Validación mediante estructuras condicionales.
- Retroalimentación visual según el resultado obtenido.

### Contraseñas válidas

| Password | Resultado |
|----------|-----------|
| 911 | password 1 correcto |
| 714 | password 2 correcto |
| Cualquier otro | password incorrecto |

### Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
